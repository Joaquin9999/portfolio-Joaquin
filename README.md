# portfolio-Joaquin

Portafolio personal en **Astro** de Joaquín Castaño Trujillo (**Neyi21**), con enfoque en ciberseguridad, CTFs, writeups y proyectos técnicos.

## Contenido

- Home con perfil profesional
- Sección de resultados CTF (CTFtime + fallback)
- Sección de writeups
- Sección de proyectos
- Experiencia y contacto

## Stack

- [Astro](https://astro.build/)
- HTML + CSS + JavaScript

## Requisitos

- Node.js 18+ (recomendado 20+)
- npm

## Instalación

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

Abre `http://localhost:4321`.

## Build de producción

```bash
npm run build
```

## Preview local de producción

```bash
npm run preview
```

```text
src/

  lib/
  pages/
  styles/
public/
```
